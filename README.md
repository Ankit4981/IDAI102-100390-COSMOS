## 🌌 What is COSMOS?

> *A production-grade aerospace analytics platform that replicates the visual language of real mission control systems — featuring live physics simulation, 196 historical space missions, and a Hubble Deep Field animated background.*

**COSMOS Mission Control Dashboard** brings the look and feel of **NASA**, **SpaceX**, **ESA**, and **ISRO** mission control rooms to your browser. Built entirely in Python with Streamlit, it combines real historical mission data with an interactive rocket physics engine — all wrapped in a dark futuristic UI.

## ⚙️ Physics Engine

Real-time **Euler integration** of the Tsiolkovsky rocket equation with exponential atmospheric drag:

```
F_net  = F_thrust − m·g − F_drag        (Net force, Newton's 2nd Law)
a      = F_net / m_total                 (Acceleration)
ρ(h)   = 1.225 · exp(−h / 8500)         (Atmospheric density model, kg/m³)
F_drag = ½ · ρ · Cᵈ · A · v²           (Aerodynamic drag, N)
m(t)   = m_dry + m_payload + m_fuel(t)  (Total mass, decreasing with burn)
```

<div align="center">

| Parameter | Min | Max | Default |
|:---:|:---:|:---:|:---:|
| ⚡ Thrust | 500 kN | 12,000 kN | **4,200 kN** |
| ⚖️ Dry Mass | 5,000 kg | 120,000 kg | **28,000 kg** |
| 📦 Payload | 500 kg | 60,000 kg | **12,000 kg** |
| ⛽ Fuel | 5,000 kg | 600,000 kg | **180,000 kg** |
| 🌬️ Drag Coefficient | 0.10 | 1.60 | **0.38** |
| 🔥 Burn Rate | 50 kg/s | 2,500 kg/s | **480 kg/s** |
| ⏱️ Time Step | 0.10 s | 2.00 s | **0.50 s** |

## 🏗️ Application Architecture

```
cosmos_app.py  (Entry Point)
│
├── inject_space_background()    ←  Canvas animation  (global, all pages)
│
├── show_login()                 ←  SHA-256 auth · role selection · telemetry chips
├── show_loading()               ←  Animated boot-sequence
├── show_landing()               ←  Cinematic mission stats entry page
│
└── show_dashboard()
    ├── Sidebar (filters + navigation)
    ├── Page 1 · Mission Control Overview  — KPIs, charts, heatmap
    ├── Page 2 · Mission Data Analytics   — 5 Plotly visualizations
    ├── Page 3 · Rocket Launch Simulation — run_simulation() → Euler loop
    ├── Page 4 · Scientific Insights      — yield analysis, insight cards
    ├── Page 5 · Dataset Explorer         — table, CSV export, stats
    └── Page 6 · About Project            — math model, docs, install guide
```
